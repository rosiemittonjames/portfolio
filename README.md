## Cancer Circle 
The biggest project I've ever worked on that I'm most proud of is Cancer Circle. I created this in my final year of University as my final year project. 

Cancer Circle is a social networking website with the purpose of helping reduce loneliness with those who've been affected by cancer, whether they're a patient themselves, a family member, a caregiver, or a friend. 

### Networking 
Networking was a key part of this project. Dr Stewart Manning, a qualified doctor who is now the chairman of the Yorkshire Cancer Community, mentored me throughout my project. As a result of this, I went to three Patient Panels (user feedback sessions) where I spoke to NHS employees, Macmillan employees, cancer charity CEOs, and cancer patients themselves.

### Process
#### Research:
- Speaking to professionals and researchers
- Speaking to cancer patients
- Attending patient panels and pitching my ideas, designing, and testing the UI and UX design

#### Design
- Wireframing on Figma 
- Designed the logo
- Created a colour palette
- Designed the whole platform on Figma 
  
#### Coding
- I used languages: HTML, CSS, JavaScript, and PHP
- I also used MySQL to collect user data
- I used Visual Studio Code
  
#### Results

- 100% of people asked would have used Cancer Circle in their journey
- 100% of people thought Cancer Circle was a good idea

### Pet Adoption API 

In this assignment, I decided to create a pet adoption API. It is a Node.js and Express application that is connected to a MYSQL database. The purpose behind this is to it allow users, such as staff, who want to manage the information about the pets which are up for adoption.

User can:
- See which animals are up for adoption.
- Add any new pets that are up for adoption.
- Delete pets who have gone to a new home.
- See a filtered version of animals based on whether they're interested in seeing dogs or cats.
- This is useful in the real world because at an adoption center, members of staff need to manage available pets quickly and efficiently.

This API could serve as the backend for a website that has an updated list of the animals that are up for adoption to increase the chances of the pets being adopted.
